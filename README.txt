Hotel Jeevak - Final (auto-detect base path)

Instructions:
1. Download and unzip into your XAMPP 'htdocs' folder.
2. Folder name can be anything (auto-detect built-in).
3. Start Apache in XAMPP control panel.
4. Open in browser: http://localhost/your-folder-name/
5. Pages included: index.php, about.php, rooms.php, facilities.php, restaurant.php, gallery.php, contact.php, booking.php
6. Replace images in assets/images/ with your real photos keeping the same filenames if you want to update visuals.
